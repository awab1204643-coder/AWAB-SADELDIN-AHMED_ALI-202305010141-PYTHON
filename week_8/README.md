Purpose

This program generates a food delivery receipt.

Tech Stack
Python 3
How to Run
python main.py
Demo

Record a short screen recording showing:

Running main.py
Entering customer details

Expected Output

If you enter:

Customer Name: Izzad
Food Ordered: Cake
Quantity: 2
Price: 3
Delivery: Y

You should get something like:

=== Customer Information ===

========== RECEIPT ==========
Customer : Izzad
Food     : Cake
Quantity : 2
Price    : RM 3.00
-----------------------------
Subtotal           : RM 6.00
Service Charge(5%) : RM 0.30
Delivery Charge    : RM 5.00
-----------------------------
TOTAL              : RM 11.30

This matches the expected output shown in your tutorial sheet.
